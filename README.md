# PitWall 🏎️

PitWall is a Formula 1 analytics dashboard built with React, TypeScript, and the OpenF1 API.

## Features

- Dynamic driver list from OpenF1
- Driver battle selector
- Team-colored driver cards
- Driver images and race numbers
- Responsive dark F1-inspired UI

## Tech Stack

- React
- TypeScript
- Vite
- OpenF1 API
- CSS

## Future Plans

- Lap time comparison
- Race weekend dashboard
- Session selector
- Weather data
- Charts for driver performance
